The "Hybrid Brain" Orchestration

Why choose between privacy and power? Our dual-AI architecture leverages the best of local and cloud processing for a seamless, secure learning environment.

Local Edge

Gemma 3 via Ollama

  • Privacy-focused on-device PDF parsing.
  • Lightning-fast JSON exercise structuring.
  • Zero latency for initial document processing.

Cloud Power

Gemini 1.5 Flash

  • Expert tutor for deep reasoning.
  • Nuanced linguistic feedback and error analysis.
  • Context-aware suggestions for complex language constructs.

Software Mansion Stack

Built for uncompromising performance and visual fidelity. We combine the React Native ecosystem's best tools to deliver a truly native feel on any platform.

SWM Reanimated

60fps fluid state transitions. Worklet-driven animations that bypass the JS thread entirely for buttery-smooth interactions.

SWM Skia

High-performance visual feedback markers. 2D graphics library providing rich, declarative canvas drawing APIs.

Expo

True cross-platform execution. Write once, deploy to Web, iOS, and Android with unified native modules and navigation.

Gemini AI

Integrated directly into the UI layer to provide contextual, nuanced linguistic feedback exactly when the user needs it.
